# flutter-omniscient-chatbot

A Flutter-based Omniscient Chatbot with a modern dark-themed mobile interface, integrated with TogoLM.

## Setup
1. Open this project in Android Studio or VS Code.
2. Run `flutter pub get` to install dependencies.
3. Provide your TogoLM API Key in the UI to start chatting.

## TogoLM Integration
This app queries `https://api.togolm.com/v1/query` using the provided prompt and API key.
